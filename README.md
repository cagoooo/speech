# 🎤 國小國語演說比賽訓練平台

> AI 智慧陪練 · 即時語音分析 · 完整評分 — 專為國小學生設計的演說訓練神器

🌐 **線上版**：[cagoooo.github.io/speech](https://cagoooo.github.io/speech/)

---

## ✨ 功能特色（v2.2，共 28 項功能）

### 🎯 核心訓練工具
- **隨機抽題系統**：56 道題目、7 大分類，模擬正式比賽抽題
- **雙階段計時器**：準備 10 分鐘倒數 ＋ 演說 4–5 分鐘計時
- **計時扣分系統**：自動計算超時扣分，完整分數卡片呈現
- **即時語音辨識**：Chrome/Edge 瀏覽器即時語音轉文字
- **字數統計**：即時字數計算 ＋ 演說時長預估

### 🤖 AI 輔助功能（Google Gemini API）
- **📋 演說大綱**：結構化大綱，掌握演說脈絡
- **📝 完整逐字稿**：含動作/語氣舞台指示標籤
- **💡 主題建議**：多元切入角度與靈感
- **✨ 名言金句**：精選名言佳句推薦
- **🎭 AI 即時評析**：語速、停頓、用詞多樣性分析
- **💌 家長通知信**：一鍵生成練習摘要信件
- **😊 情感色彩分析**：激勵感、溫情感、說服力、邏輯感四維分析

### 🎨 UI/UX 體驗
- **進度條 UI**：五階段進度條（初始化 → 傳送 → 生成 → 渲染 → 完成）
- **Markdown 精美渲染**：標題、列表、引用、表格完整支援
- **動作/語氣標籤**：🎭 紫色漸層標籤，區分口述與表演指引
- **四種模式 PDF 列印**：每種模式獨立色彩主題與標題，一眼辨識
- **社群分享預覽**：OG / Twitter Card 完整卡片支援

### 📦 其他功能
- 題庫自訂管理（新增 / 刪除）
- 題目收藏標記（⭐）
- 準備時間自訂（1–15 分鐘）
- 計時音效提示（Web Audio API）
- 資料 localStorage 本地儲存
- GitHub Pages 自動部署

## 🚀 使用方式

### 線上使用（推薦）
直接開啟 👉 [cagoooo.github.io/speech](https://cagoooo.github.io/speech/)

### 本機使用
1. 下載或 clone 本專案
2. 直接用瀏覽器開啟 `index.html`
3. （可選）填入 Google Gemini API Key 啟用 AI 功能

### AI 功能設定
1. 前往 [Google AI Studio](https://aistudio.google.com/) 取得免費 API Key
2. 在平台頂部「Gemini API 金鑰」欄位貼上
3. 即可使用所有 AI 輔助功能（無 API Key 仍可使用範例模式）

## 🛠️ 技術架構

- **前端**：純 HTML/CSS/JavaScript（單一檔案，零依賴）
- **AI**：Google Gemini API（gemini-2.0-flash / gemini-1.5-pro）
- **語音**：Web Speech API（瀏覽器原生）
- **音效**：Web Audio API（合成提示音）
- **部署**：GitHub Actions → GitHub Pages
- **儲存**：localStorage（本地端）
