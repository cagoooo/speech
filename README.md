## 🚀 使用方式

### 線上使用（推薦）
直接開啟 👉 [cagoooo.github.io/speech](https://cagoooo.github.io/speech/)

### 本機使用
1. 下載或 clone 本專案
2. 直接用瀏覽器開啟 `index.html`
3. （可選）填入 Google Gemini API Key 啟用 AI 功能

### 雲端同步設定（可選）
1. 點擊右上角「使用 Google 登入」
2. 授權後練習紀錄自動同步到雲端
3. 切換到「我的歷史」查看完整練習紀錄

### AI 功能設定
1. 前往 [Google AI Studio](https://aistudio.google.com/) 取得免費 API Key
2. 在平台頂部「Gemini API 金鑰」欄位貼上
3. 即可使用所有 AI 輔助功能（無 API Key 仍可使用範例模式）

## 🛠️ 技術架構

- **前端**：純 HTML/CSS/JavaScript（單一檔案，零依賴）
- **AI**：Google Gemini API（預設 `gemini-2.5-flash-lite`）
- **後端**：Supabase（PostgreSQL + Auth + RLS）
- **登入**：Google OAuth（透過 Supabase Auth）
- **語音**：Web Speech API（瀏覽器原生）
- **音效**：Web Audio API（合成提示音）
- **部署**：GitHub Actions → GitHub Pages
- **儲存**：Supabase（雲端）／ localStorage（本地 fallback）
